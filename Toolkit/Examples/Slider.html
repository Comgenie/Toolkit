Toolkit

TK.Slider

Slider component

Also available as form element using type 'slider'.

{
    _: TK.Slider,
    Min: 0,
    Max: 100,
    Data: 75
}

Slider with steps and text

{
    _: TK.Slider,
    Min: 0,
    Max: 100,
    StepSize: 20,
    ShowSteps: true,
    Data: 40,
    TextAfter: "40 %",
    onchange: function() {
        this.TextAfter = this.Data + " %";
    }
}
