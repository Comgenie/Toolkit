Toolkit Javascript Framework

A simple toolkit for your everyday web development. All components are built for high compatiblity and extensibility.

<script src="https://toolkit.comgenie.com/minified/toolkit.js"></script>

Optional CSS:

<link href="https://toolkit.comgenie.com/minified/style.css" rel="stylesheet" />
